# Skald v1 (archived)

This is the original Skald implementation (React + Tailwind + Lexical/Monaco,
multi-theme "winamp skin" era). It was archived when the app was rewritten
around the editor-native design system that now lives at the repository root.

It is kept for reference only — it is not built, tested, or maintained.
To run it anyway: `cd archive/skald-v1 && npm install && npm run electron:dev`.
